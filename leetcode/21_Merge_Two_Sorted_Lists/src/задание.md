# 21. Merge Two Sorted Lists
Easy

You are given the heads of two sorted linked lists list1 and list2.

Merge the two lists into one sorted list. The list should be made by splicing together the nodes of the first two lists.

Return the head of the merged linked list.

Example 1:

Input: list1 = [1,2,4], list2 = [1,3,4]
Output: [1,1,2,3,4,4]
Example 2:

Input: list1 = [], list2 = []
Output: []
Example 3:

Input: list1 = [], list2 = [0]
Output: [0]

Constraints:

The number of nodes in both lists is in the range [0, 50].
-100 <= Node.val <= 100
Both list1 and list2 are sorted in non-decreasing order.

Перевод условия
Даны головы (head) двух отсортированных связных списков: list1 и list2.

Нужно объединить их в один отсортированный список, соединяя узлы исходных списков (создавать новые узлы не требуется, можно переиспользовать существующие).

Вернуть голову (head) объединённого списка.

Примеры
Вход: list1 = [1,2,4], list2 = [1,3,4]
Выход: [1,1,2,3,4,4]

Вход: list1 = [], list2 = []
Выход: []

Вход: list1 = [], list2 = [0]
Выход: [0]

Ограничения
Количество узлов в каждом списке: от 0 до 50

Значения узлов: от -100 до 100

Оба списка отсортированы по неубыванию (т.е. каждый следующий узел ≥ предыдущего)

Пояснение задачи
У тебя есть два уже отсортированных связных списка.
Нужно слить их в один, также отсортированный, но без создания новых узлов — просто переназначая указатели next.

Это классическая задача на двух указателях (как при слиянии в сортировке слиянием).

Идея решения
Создаём фиктивный узел (dummy), чтобы упростить обработку начала списка.

Идём двумя указателями по list1 и list2, сравнивая текущие значения.

Меньший узел прицепляем к результату и сдвигаем указатель в соответствующем списке.

Когда один из списков заканчивается, прицепляем остаток второго.

Пример работы
list1: 1 → 2 → 4
list2: 1 → 3 → 4

Сравниваем 1 и 1 → берём из list1
Сравниваем 2 и 1 → берём из list2
Сравниваем 2 и 3 → берём из list1
Сравниваем 4 и 3 → берём из list2
Сравниваем 4 и 4 → берём из list1
Остаток list2 → 4

Результат: 1 → 1 → 2 → 3 → 4 → 4

Сложность
Время: O(n + m), где n, m — длины списков

Память: O(1) (без учёта входных данных) — только несколько указателей

Если нужно, могу написать код на Python (итеративный или рекурсивный) с подробными комментариями.
